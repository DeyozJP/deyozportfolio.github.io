May, 2023

Click Through Rate Prediction

In this project, we used 14 days of mobile ad data, available in Kaggle, to develop and evaluate models that predict whether ads shown to the user will click or not, eventually determining CTR. The main feature of this project is to show how to transform the categorical variables that have many values to make one-hot encoding an inappropriate method of transformation. Another highlight of this project is how to determine the optimum threshold for classification and present the trade-offs between recall and precision. Also, model evaluation techniques such as Receiver Operating Curve, Area Under the Curve, Cumulative Gain Curve, and Lift Curve are applied to evaluate the model performance.

We used Gradient Boost, XG Boost, Logistic Regression, Naïve Bayes, and Random Forest. Based on the ROC and AUC, Gradient Boost and XG Boost outperformed other models. Based on the resource requirements, we selected XG Boost. However, the recall rate, which is an important metric in this use case, was low initially. We increased the recall rate to 87 percent by reducing the threshold to 0.15 for classification, which reduced the precision rate to 33 percent.

We further evaluated the model using cumulative gain chart and lift curve, which shows that selecting the top 20 % of the leads ranked as most likely to respond to an ad will contain about 55% of the actual click. The expected baseline number without using the model would only be 20%. Therefore, the model adds business value.
